# ⚡ Automated Electricity Consumption Monitoring System

## 📌 Overview
This project is an end-to-end Data Engineering pipeline that automates the collection, processing, and visualization of electricity consumption data.

It helps in monitoring power usage efficiently and provides meaningful insights through dashboards.

## 🚀 Key Features
- Automated ETL pipeline
- Data cleaning and transformation
- Workflow scheduling using Apache Airflow
- Containerized setup using Docker
- Interactive dashboard using Power BI

## 🛠 Tech Stack
- Python
- SQL
- Apache Airflow
- Docker
- Power BI

## 🔄 Pipeline Workflow
1. Data ingestion from source (CSV/API)
2. Data preprocessing using Python
3. Storage in database
4. Workflow automation using Airflow
5. Visualization using Power BI

## 📊 Results
- Processed 1M+ records
- Reduced manual effort by 80%
- Built interactive dashboard for insights
